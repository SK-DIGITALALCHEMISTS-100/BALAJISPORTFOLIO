Udemy Courses Performance

This project focuses on analyzing Udemy course data to uncover meaningful insights about course performance, pricing strategy, ratings, and learner engagement. The analysis helps understand which types of courses perform well and what factors influence popularity and revenue.

  • Analytical: Self Project
  • Completed: 29 Dec 2025
Tools & Technologies Used
  • Python
  • Jupyter Notebook
  • Pandas – data cleaning & transformation
  • NumPy – numerical operations
  • Matplotlib & Seaborn – data visualization
  • Power BI (for dashboard presentation concept)
Data Cleaning & Preparation
  • Removed missing and duplicate values
  • Standardized column names
  • Converted data types for numerical analysis
  • Handled outliers in price and subscriber counts
  • Created derived columns for better insights
Exploratory Data Analysis (EDA)
  • Distribution of free vs paid courses
  • Relationship between price and subscribers
  • Impact of ratings on course popularity
  • Category-wise course count and performance
  • Comparison of course levels (Beginner vs Advanced)
  • Top courses by subscribers and ratings
Conclusion

This project highlights my ability to transform raw educational data into actionable insights using Python and visualization techniques. It reflects real-world data analytics workflows and business-oriented thinking.
