Dine Desk is a restaurant management and food ordering platform designed to enhance the dining experience by enabling digital table reservations, order tracking, and menu browsing. As part of an internship project within a company environment, the focus was on developing the foundational modules of the application, specifically the login, signup, and home page functionalities.

The login and signup modules ensure secure user authentication and account management, allowing new users to register and existing users to access personalized features. These modules were developed with form validation, session handling, and basic security protocols to ensure a smooth and reliable user experience.

The home page serves as the entry point for the application, offering a clean, responsive design with navigation to core features such as menu display, table booking, and user profile access. This page was built to align with modern UI/UX principles and optimized for both desktop and mobile devices.

This project demonstrates the implementation of key frontend and backend concepts in a real-world setting, contributing to the base framework of a scalable restaurant application. It showcases proficiency in web development, user interface design, and secure authentication practices.
